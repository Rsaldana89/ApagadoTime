ApagadoTime

ApagadoTime is a Python application that allows you to schedule the shutdown of your device. You can specify the time at which you want your computer to automatically shut down.

Prerequisites

Python 3.x installed on your system. You can download it from python.org.
Tkinter library. If you don't have Tkinter installed, you can install it using the following command:
Copy code
pip install tkinter
Usage Instructions
Clone or download this repository to your computer.

Open a terminal and navigate to the location of the "apagado.py" file.

Run the following command to start the application:

Copy code
python apagado.py
The ApagadoTime application window will appear.

Click the "INGRESA EN CUANTO TIEMPO DEBE APAGARSE LA PC" button to display the configuration options.

Enter the number of hours and minutes in the respective fields.

Click the "Apagar" button to schedule the shutdown of your device.

To cancel the scheduled shutdown, click the "Cancelar Apagado" button.

The "TAMBIEN PUEDES CANCELAR EL APAGADO" label will display a confirmation message when the shutdown is canceled.
